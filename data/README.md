# Data Directory

This directory stores the vector database and any generated data files.

## Files

### Vector Database
- `Task_Theory_Part_1_DB.json` - The vector database containing embedded course content and code examples

This file is generated by running:
```bash
python scripts/build_database.py
```

### Source Data
Source data files should remain in the project root:
- `Task_Theory_Part_1.txt` - Course content
- `Task_Theory_Part_2.txt` - Additional course content
- `Code_QnA.xlsx` - Code examples

## Regenerating the Database

If you update the source materials, rebuild the vector database:

```bash
python scripts/build_database.py --input Task_Theory_Part_1.txt --code-examples Code_QnA.xlsx --output data/Task_Theory_Part_1_DB.json
```

## Git Tracking

The `.gitignore` is configured to allow JSON files in this directory while excluding result files like `results.json` from the root.
